linux安装安卓控制工具，QtScrcpy

Jul 09, 2024

介绍

  • 下载地址
  • 文档地址

安装

创建文件夹并解压

1
2

sudo mkdir /opt/QtScrcpy
sudo unzip QtScrcpy-ubuntu-22.04-gcc_64.zip -d /opt/QtScrcpy/

创建软连接

1

sudo ln -s /opt/QtScrcpy/output/x64/Release/QtScrcpy /usr/bin/qtscrcpy

创建desktop，让开始菜单图标/全局搜索/桌面图标可用

1

sudo vim /usr/share/applications/qtscrcpy.desktop

将如下内容粘贴

1
2
3
4
5
6
7
8
9

[Desktop Entry]
Encoding=UTF-8
Name=QtScrcpy
Exec=qtscrcpy %U
Icon=qtscrcpy
Keywords=qtscrcpy
Categories=Network;
Type=Application
Terminal=false

预览

使用

使用usb连接手机，开启usb调试，QtScrcpy会自动刷新出设备，点击一键USB连接

常见问题

连接无反应

尝试在终端启动 QtScrcpy

1

qtscrcpy

启动后再次连接，查看终端报错信息

报错1：

The server version (2.4) does not match the client (2.1.1)

解决：

使用scrcpy-server 2.1.1替换默认的scrcpy-server

scrcpy-server 2.1.1下载地址

使用这个命令替换默认的scrcpy-server

1

sudo cp scrcpy-server-v2.1.1 /opt/QtScrcpy/output/x64/Release/scrcpy-server

再次尝试启动

Qtscrcpy
