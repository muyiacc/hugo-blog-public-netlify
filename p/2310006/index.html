Debian12下的软件清单

Oct 06, 2023

本文列举安装并使用的的一些软件，基于debian12，debain12有很大的更新，使用wayland替代了x11，本文编辑时，可能好多软件没适配，体验感稍差。

本文将持续更新

Debian 官方指南：Debian 系统中可用的软件

浏览器

Firefox ESR

默认预装，感觉可以，更新频率比较慢

Firefox

更新速度较快，有很多Firefox ESR没有的新功能

Google chrome

Brave

https://brave.com/linux/

sudo apt install curl

sudo curl -fsSLo /usr/share/keyrings/brave-browser-archive-keyring.gpg https://brave-browser-apt-release.s3.brave.com/brave-browser-archive-keyring.gpg

echo "deb [signed-by=/usr/share/keyrings/brave-browser-archive-keyring.gpg] https://brave-browser-apt-release.s3.brave.com/ stable main"|sudo tee /etc/apt/sources.list.d/brave-browser-release.list

sudo apt update

sudo apt install brave-browser

下载工具

Free Download Manager

截止本文，我觉得是linux上最好的下载工具

更新：推荐使用 gopeed

gopeed

https://github.com/GopeedLab/gopeed

XDM

很混乱，感觉好久没怎么更新了，体验感较差

社交

QQ

通过官网下载安装即可

微信

推荐使用deepin-wechat，分辨率有问题，能尽量避免不用就不用吧，体验感很差

官方已推出linux测试版，通过这个仓库可以下载：

https://github.com/lovechoudoufu/wechat_for_linux

播放器

VLC

mpv

上手有一定难度，配置文件可备份，安装后即可使用

个人配置仓库：https://github.com/muyiacc/mpv-config

zyplayer

可以用来看影视

音乐

LX Music

推荐，跨平台使用，可下载，但是界面来说，我感觉一般，播放没有列表选择，本地音乐支持不好，主要用来听网络上的歌曲

可以使用网易云的歌单

Spotify

官方教程：https://www.spotify.com/us/download/linux/

1
2

curl -sS https://download.spotify.com/debian/pubkey_6224F9941A8AA6D1.gpg | sudo gpg --dearmor --yes -o /etc/apt/trusted.gpg.d/spotify.gpg  
echo "deb http://repository.spotify.com stable non-free" | sudo tee /etc/apt/sources.list.d/spotify.list

1

sudo apt-get update && sudo apt-get install spotify-client

去广告脚本

1

bash <(curl -sSL https://spotx-official.github.io/run.sh)

Amberol

这是一款本地播放器，颜值高，Kde系通过 Flathub 安装

1

flatpak install flathub io.bassi.Amberol

编程

Jetbrains Toolbox

推荐，使用它安装其他jetbrains家族的软件，可以解决分辨率的问题，而且安装起来也比较方便。

而且，学生免费

Navicat

好看是好看，就是激活太麻烦了，不如用 Jetbrains 的Data Grip

更新：Navicat推出了 lite版本，个人免费使用
