aria2下载工具安装，并安装前端界面webui-aria2

Dec 01, 2023

1. 概述

aria2 is a lightweight multi-protocol & multi-source command-line download utility. It supports HTTP/HTTPS, FTP, SFTP, BitTorrent and Metalink. aria2 can be manipulated via built-in JSON-RPC and XML-RPC interfaces.

2. 下载安装

2.1. 下载

下载地址：https://github.com/aria2/aria2/releases/

最新版Windows 64版本：点我直接下载

2.2. 安装

解压到指定的路径，建议新建 aria2 文件夹，放入即可，把目录添加到环境变量。

2.3. 配置

在 aria2 目录下新建以下文件：

  • aria2.conf （配置文件）
  • aria2.log （日志文件，空文件即可）
  • aria2.session （保存下载进度的会话文件，空文件即可）
  • Run.bat （启动脚本）
  • HideRun.vbs （无界面启动脚本）

# 文件的保存路径(可使用绝对路径或相对路径), 默认: 当前启动位置
dir=E:\Downloads
# 日志
log=D:\Program Files\aria2\aria2.log
# 日志级别：不要缺这句，否则日志文件的大小会爆炸
log-level=warn
# 从会话文件中读取下载任务
input-file=D:\Program Files\aria2\aria2.session
# 在Aria2退出时保存`错误/未完成`的下载任务到会话文件
save-session=D:\Program Files\aria2\aria2.session
# 定时保存会话, 0为退出时才保存, 需1.16.1以上版本, 默认:0
save-session-interval=30

enable-rpc=true
pause=false
rpc-allow-origin-all=true
rpc-listen-all=true
rpc-save-upload-metadata=true
rpc-secure=false

Run.bat

1

aria2c.exe --conf-path=aria2.conf

HideRun.vbs： ！！特别注意符号

1

WScript.CreateObject("WScript.Shell").Run """D:\Program Files\aria2\Run.bat""",0

对 HideRun.vbs 右键创建快捷方式，放入启动目录

1

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp

3. webui-aria2

3.1 概述

The aim for this project is to create the worlds best and hottest interface to interact with aria2. aria2 is the worlds best file downloader, but sometimes the command line brings more power than necessary. The project was initially created as part of the GSOC scheme, however it has rapidly grown and changed with tremendous support and feedback from the aria2 community.

简单来说就是一个web ui界面

地址：https://github.com/ziahamza/webui-aria2

3.2 下载

进入 aria2 目录 D:\Program Files\aria2

使用git克隆项目

1

 git clone --depth 1 https://github.com/ziahamza/webui-aria2.git

3.3 启动

  1. 启动 aria2c
1

aria2c --enable-rpc --rpc-listen-all --conf-path="D:\Program Files\aria2\aria2.conf"
  1. 启动 webui-aria2

进入克隆的项目文件夹，其下有个 doc文件夹，进入，双击 index.html文件即可运行。
