基本设置

灯

https://www.bilibili.com/read/cv21700714/

玩客云这设备有个红、绿、蓝三色led灯，在刷入armbian系统以后，这个灯的颜色就会显示异常，往往是一直显示红色。

如果要自动动手调整led灯的颜色，控制命令如下（需要root用户执行）

1
2
3

echo 1 > /sys/class/leds/onecloud:blue:alive/brightness
echo 1 > /sys/class/leds/onecloud:green:alive/brightness
echo 1 > /sys/class/leds/onecloud:red:alive/brightness

echo 1是开，echo 0就是关。现在可以组合出多种颜色，蓝绿、红蓝、红绿、红绿蓝。😄

但这个设置重启后失效。╮(╯▽╰)╭

静态ip

查看当前网络连接信息

1

nmcli connection show

设置静态ip

1
2
3
4

nmcli connection modify "Wired connection 1" ipv4.addresses 192.168.68.9/24
nmcli connection modify "Wired connection 1" ipv4.gateway 192.168.68.1
nmcli connection modify "Wired connection 1" ipv4.dns 223.5.5.5
nmcli connection modify "Wired connection 1" ipv4.method manual

重新启动网络连接：

1
2

nmcli connection down "Wired connection 1" 
nmcli connection up "Wired connection 1"

检查网络配置，确保静态 IP 已成功应用：

1

nmcli connection show "Wired connection 1"

# 下载
mkdir -p /opt/alist
wget https://github.com/alist-org/alist/releases/download/v3.37.4/alist-linux-arm-7.tar.gz
tar -zxvf alist-linux-arm-7.tar.gz -C /opt/alist

# 安装启动
cd /opt/alist
chmod +x alist
./alist server # 启动会前台阻塞
# 新开一个终端设置密码
./alist admin set 密码


# 安装服务
vim /usr/lib/systemd/system/alist.service
# 填入一下内容
[Unit]
Description=alist
After=network.target
 
[Service]
Type=simple
WorkingDirectory=/opt/alist
ExecStart=/opt/alist/alist server
Restart=on-failure
 
[Install]
WantedBy=multi-user.target

wg-esay

异地组网工具wireguard，wg-easy是简化wireguard部署过程的开源项目，使用docker部署，简单快速上手。

wireguard 是虚拟加密网络（VPN）,它可以使异地的设备组成一个局域网，换句话说就是可以在外地可以像在家中一样访问家中的设备，ip地址都可以不变。

源码：https://github.com/wg-easy/wg-easy

部署方式：

  -e PORT=51821 \

  -v ~/.wg-easy:/etc/wireguard \
  -p 51820:51820/udp \
  -p 51821:51821/tcp \
  --cap-add=NET_ADMIN \
  --cap-add=SYS_MODULE \
  --sysctl="net.ipv4.conf.all.src_valid_mark=1" \
  --sysctl="net.ipv4.ip_forward=1" \
  --restart unless-stopped \
  ghcr.io/wg-easy/wg-easy
