常见 Vimrc 配置

Vimrc 文件（通常位于 ~/.vimrc 或 ~/.vim/vimrc）是 Vim 的配置文件，用于自定义 Vim 的行为和外观。 以下是一些常见的配置选项，可以根据个人喜好进行调整：

1. 基本设置：

  • 设置行号：

    1
2

    set number      " 显示行号
set relativenumber " 显示相对行号 (当前行显示0)

  • 设置缩进：

    1
2
3
4
5
6

    set tabstop=4   " Tab 键宽度为 4 个空格
set shiftwidth=4  " 自动缩进宽度为 4 个空格
set expandtab   " 将 Tab 键转换为空格
set autoindent  " 自动缩进
set smartindent " 智能缩进
set cindent     " C风格缩进

  • 设置搜索：

    1
2
3
4

    set hlsearch    " 高亮显示搜索结果
set incsearch   " 实时搜索
set ignorecase  " 搜索时忽略大小写
set smartcase   " 如果搜索包含大写字母，则区分大小写

  • 设置光标行/列高亮:

    1
2

    set cursorline  " 高亮显示光标所在行
set cursorcolumn " 高亮显示光标所在列

  • 设置备份和交换文件：

    1
2
3
4
5
6

    set backup      " 创建备份文件
set writebackup " 写文件时先备份
set backupdir=~/.vim/backup " 备份文件存放目录
set swapfile    " 创建交换文件 (用于恢复意外关闭的文件)
set directory=~/.vim/swap  " 交换文件存放目录
set noswapfile  " 禁止创建交换文件 (谨慎使用)

  • 设置编码：

    1
2
3

    set encoding=utf-8      " 设置文件编码为 UTF-8
set fileencoding=utf-8  " 设置文件保存编码为 UTF-8
set fileencodings=utf-8,gbk,ucs-bom,utf-16,latin1 " 自动检测文件编码

  • 设置兼容性：

    1

    set nocompatible " 关闭兼容模式，使用 Vim 增强功能

  • 设置鼠标：

    1

    set mouse=a     " 启用鼠标支持 (在终端和 GUI 中)

  • 设置语法高亮：

    1
2

    syntax on       " 启用语法高亮
colorscheme solarized " 设置配色方案 (需要安装)

  • 设置自动补全：

    1
2

    set wildmenu    " 启用增强型命令行补全
set wildmode=longest:list,full " 设置补全模式

  • 设置历史记录：

    1

    set history=1000 " 保存 1000 条历史命令

  • 设置命令窗口高度：

    1

    set cmdheight=2  " 命令行窗口高度为 2 行

  • 设置显示空格和制表符:

    1
2

    set list
set listchars=tab:>\ ,trail:-,eol:$

2. 界面设置：

  • 设置状态栏：

    1
2

    set laststatus=2 " 总是显示状态栏
set statusline=%F%m%r%h%w\ [FORMAT=%{&ff}]\ [TYPE=%Y]\ [POS=%l,%v][%p%%]\ %{strftime("%Y-%m-%d\ %H:%M:%S")}

  • 设置滚动：

    1
2

    set scrolloff=5  " 光标距离屏幕顶部/底部至少 5 行
set sidescrolloff=10 " 光标距离屏幕左侧/右侧至少 10 列

  • 设置隐藏菜单和工具栏：

    1
2

    set guioptions-=m " 隐藏菜单栏
set guioptions-=T " 隐藏工具栏

  • 设置窗口分割：

    1
2

    set splitbelow " 新窗口在下方/右侧打开
set splitright

3. 快捷键映射 (Mappings):

  • 取消高亮搜索：

    1

    nnoremap <leader><leader> :nohlsearch<CR>

    ( <leader> 默认是反斜杠 \， 可以通过 let mapleader="," 修改)

  • 快速保存：

    1

    nnoremap <leader>w :w!<CR>

  • 快速退出：

    1

    nnoremap <leader>q :q!<CR>

  • 快速切换窗口：

    1
2
3
4

    nnoremap <C-h> <C-w>h
nnoremap <C-j> <C-w>j
nnoremap <C-k> <C-w>k
nnoremap <C-l> <C-w>l

  • 复制到系统剪贴板： (需要 Vim 支持 +clipboard 特性)

    1
2
3
4

    vnoremap <leader>y "+y
nnoremap <leader>Y "+Y
nnoremap <leader>p "+p
nnoremap <leader>P "+P

  • 打开新的标签页：

    1

    nnoremap <leader>t :tabnew<CR>

    " vim-plug 安装 (如果未安装)
if empty(glob('~/.vim/autoload/plug.vim'))
  silent !curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

5. 特定语言/文件类型的设置:

  • Python:

    1

    autocmd FileType python setlocal tabstop=4 shiftwidth=4 expandtab autoindent fileformat=unix

  • JavaScript:

    1

    autocmd FileType javascript setlocal tabstop=2 shiftwidth=2 expandtab autoindent

  • Markdown:

    1

    autocmd FileType markdown setlocal wrap linebreak textwidth=79

解释:

  • set 命令用于设置 Vim 的选项。
  • nnoremap 命令用于创建非递归的普通模式快捷键映射。
  • vnoremap 命令用于创建非递归的可视模式快捷键映射。
  • autocmd 命令用于定义自动命令，在特定事件发生时执行命令。
  • <leader> 是一个特殊的键，默认是反斜杠 \，可以自定义。
  • <CR> 代表回车键。
  • Plug 命令是 vim-plug 插件管理器提供的命令，用于安装插件。

如何使用:

  1. 创建或编辑 vimrc 文件: 打开终端，输入 vim ~/.vimrc 或 vim ~/.vim/vimrc。
  2. 复制和粘贴配置: 将上面的配置选项复制并粘贴到 vimrc 文件中。
  3. 保存并退出: 按下 Esc 键，输入 :wq 并按下回车键。
  4. 重新加载 vimrc 文件: 在 Vim 中输入 :source ~/.vimrc 或 :source ~/.vim/vimrc，或者重启 Vim。

重要提示:

  • 备份你的 vimrc 文件: 在修改之前，最好备份你的 vimrc 文件，以防出现问题。
  • 逐步添加配置: 不要一次性添加所有配置，最好逐步添加，并测试每个配置是否有效。
  • 查阅 Vim 文档: Vim 的文档非常全面，可以查阅文档了解更多信息。 输入 :help 在 Vim 中打开帮助文档。
  • 使用插件管理器: 强烈建议使用插件管理器来管理你的插件，例如 vim-plug、Vundle 或 Pathogen。
  • 根据个人喜好进行调整: 这些只是一些常见的配置选项，你可以根据自己的喜好进行调整。

通过配置 vimrc 文件，你可以定制 Vim 的行为和外观，使其更适合你的工作流程，提高你的编辑效率。 祝你使用 Vim 愉快！
