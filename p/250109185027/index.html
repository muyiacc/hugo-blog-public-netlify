WebUI

设置 -> Web UI

开启Web用户界面，可以通过网页打开界面，可以远程使用，后续的 防止吸血也要用到。

配置用户名和密码

开启对本地主机客户端跳过身份验证

安装 qBittorrent-ClientBlocker

为了防止被恶意 客户端/IP 吸血，封禁某些不友好的 IP/客户端，保护 BT 网络，保护宽带。

下载地址：https://github.com/Simple-Tracker/qBittorrent-ClientBlocker/releases

解压出来之后，修改 config.json

配置 "clientURL": "",

设置为 前面开启的 Web UI 地址和端口，比如

1

"clientURL": "http://localhost:8020",

之后双击 qBittorrent-ClientBlocker.exe 即可运行。

快捷方式启动

这两个每次都分开启动，有点繁琐，可以新建快捷方式启动，点击之后可以同时启动。

桌面右键 -> 新建快捷方式，修改以下内容之后，再填入

1

C:\Windows\System32\cmd.exe /c "(tasklist | findstr qBittorrent-ClientBlocker || start D:\download_tool\qBittorrent-ClientBlocker-3.6-windows-amd64\qBittorrent-ClientBlocker.exe --hidewindow) && start D:\download_tool\qBittorrent-Enhanced-Edition\qbittorrent.exe"

需要修改的地方：

  • D:\download_tool\qBittorrent-ClientBlocker-3.6-windows-amd64\qBittorrent-ClientBlocker.exe 修改为自己的路径

  • D:\download_tool\qBittorrent-Enhanced-Edition\qbittorrent.exe 修改为自己的路径

点击 下一步，输入自己的快捷方式的名称，比如

1

qBittorrent and qBittorrent-ClientBlocker 快捷方式

点击完成即可。

如果不喜欢默认的图标，可以 鼠标移动到 快捷方式上，右键 -> 属性，找到 更改图标，设置 qBittorrent所在的图标，比如

1
