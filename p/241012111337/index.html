颜色

border-color指定边框颜色

可以使用

  • 指定的名字，比如 red、blue
  • 十六进制，比如 #ff0000
  • RGB，比如 rgb(255,0,0)
  • HSL，比如 hsl(0, 100%, 50%)
  • transparent，表示透明

<!DOCTYPE html>

<head>
<style>
p {
  border-top-style: dotted;
  border-right-style: solid;
  border-bottom-style: dotted;
  border-left-style: solid;
}
</style>
</head>
<body>

<h2>Individual Border Sides</h2>
<p>2 different border styles.</p>

</body>
</html>

除了使用特定侧边的属性，也可以只使用 border-style单个属性达到同样的效果，后续跟上不同的值有不同的效果，按照顺序为：上右下左。

  • 四个值：上右下左

  • 三个值：上右下，左跟随着右

  • 两个值：上右，下随着上，左跟随着右

  • 一个值：四边相同

}
</style>
</head>

<h2>Individual Border Sides</h2>

<p class="three">3 different border styles.</p>
<p class="two">2 different border styles.</p>
<p class="one">1 border style.</p>

</body>
</html>
