nvm，Node版本管理工具

Oct 17, 2023

概述

Node Version Manager，NodeJS 版本管理工具

Github: https://github.com/nvm-sh/nvm

快速上手

安装nvm

1

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash

查看帮助信息

1

nvm --help

查看可安装的 Node 版本

1
2

# 查看node版本
nvm ls-remote

安装默认的 Node lts 版本

1
2

# 安装lts版本
nvm install --lts # 安装默认的 lts 版本，目前是20

安装指定的 Node 版本

1

nvm install 16.20.2

查看本地已下载的 Node 信息

1

nvm ls

使用指定的 Node 版本

1
2

# 确保已经安装过
nvm use 16
